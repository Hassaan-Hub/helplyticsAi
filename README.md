## 📖 About The Project

Helplytics AI is a modern community support platform designed to encourage collaboration and knowledge sharing. Users can create help requests, offer assistance, communicate through real-time chat, and track community contributions through leaderboards and trust scores.

---

## ✨ Features

* 🔐 Authentication (Email & Google Login)
* 👤 User Profiles & Onboarding
* 📝 Create Help Requests
* 🔍 Explore & Filter Requests
* 💬 Real-Time Messaging System
* 🔔 Notifications System
* 🏆 Leaderboard & Trust Score
* 🤖 AI Insights & Suggestions
* 📊 Dashboard Analytics
* 🛠️ Admin Panel
* 🌙 Responsive Dark Theme UI
